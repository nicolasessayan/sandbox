Install CodeReef

CodeReef requires minimal dependencies (Python 2.7+ or 3.x, PIP and Git) and can be setup and used very quickly from your command line. To dig further, you can read our documentation.

1 Install

pip install codereef

2 Launch

cr start

3 Use

Components and AI solutions can be installed and configured with the specific commands mentioned in our libraries. For instance, TensorFlow (in its version 1.9.0 for CPU) can be installed as follow:

cr install tensorflow-1.9.0-cpu
